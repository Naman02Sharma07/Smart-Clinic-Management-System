# User Stories (Doctor, Patient, Admin)

## Doctor stories
- As a Doctor, I want to log in securely so that I can view my schedule.
- As a Doctor, I want to view appointments by date so that I can manage daily workload.
- As a Doctor, I want to create prescriptions for appointments so that patients receive treatment instructions.

## Patient stories
- As a Patient, I want to search doctors by name/speciality so that I can find the right specialist.
- As a Patient, I want to book appointments so that I can receive consultation.
- As a Patient, I want to view my booked appointments so that I can track upcoming visits.

## Admin stories
- As an Admin, I want to add doctors so that the clinic directory stays up to date.
- As an Admin, I want to manage records securely so that only authorized users can access data.
